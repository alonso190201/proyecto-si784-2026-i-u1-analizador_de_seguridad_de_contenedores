Aseguramiento de Calidad (QA)

Controles de Calidad y Seguridad de Software

Este sistema integra múltiples mecanismos y atributos de calidad de acuerdo con los estándares del curso Calidad y Pruebas de Software:

Análisis Estático

El sistema realiza pruebas estáticas automatizadas sobre los archivos de configuración subidos. Escanea reglas predefinidas en el backend sin necesidad de ejecutar los contenedores, minimizando riesgos de forma temprana.

Sanitización (Privacidad)

Para salvaguardar la confidencialidad de la información, el backend enmascara automáticamente contraseñas, secretos y tokens detectados reemplazándolos con asteriscos (********) antes de registrarlos o mostrarlos.

Exportación

Permite descargar reportes en formatos adaptables a distintos criterios: HTML interactivo para auditorías rápidas y SARIF (JSON estándar) para portabilidad e integración.
